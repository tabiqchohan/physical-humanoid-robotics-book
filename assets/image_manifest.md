### `assets/` Manifest

This manifest lists all image assets (diagrams, illustrations) required for the Docusaurus book.

- `chapter1_robot_arm_architecture.png`: Diagram illustrating a typical robot arm's architectural components.
- `chapter2_kinematics_diagram.png`: Diagram showing the kinematic chain of a 2-DOF robot arm with joint angles and link lengths.
- `chapter3_sensor_fusion.png`: Diagram depicting a sensor fusion architecture, showing data flow from multiple sensors to a unified perception model.
- `chapter4_gripper_design.png`: Concepts and components of various robot gripper designs (e.g., parallel, adaptive, vacuum).
- `chapter5_rl_loop.png`: Diagram illustrating the core reinforcement learning loop: Agent, Environment, State, Action, Reward.
- `chapter6_hri_workflow.png`: Workflow diagram for human-robot collaboration, showing interaction modalities and decision points.
- `chapter7_safety_zones.png`: Diagram illustrating different safety zones around a robot (e.g., collaborative, restricted, danger zones).
